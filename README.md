# 🏠 Bangalore House Price Prediction

A Machine Learning web application that predicts house prices in Bangalore based on user inputs like location, square feet, number of bedrooms, and bathrooms.


## 🚀 Features

* Predict house prices instantly
* Clean and simple web interface
* Machine Learning model trained on real dataset
* Image preprocessing using OpenCV
* Feature engineering using wavelet transform


## 🛠️ Tech Stack

* Python
* NumPy, Pandas
* Scikit-learn
* OpenCV
* PyWavelets
* Flask
* HTML, CSS, JavaScript

## ⚙️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/your-username/bhp-price-prediction.git
cd bhp-price-prediction
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the backend server

```
cd server
python app.py
```

### 4. Run the frontend

```
cd client
python -m http.server 8000
```

### 5. Open in browser

```
http://localhost:8000
```
